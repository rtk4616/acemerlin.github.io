Windows 10 Bash & Linux Subsystem & Spacemacs Setup

Posted on 2018-04-09   |   In 杂谈

I’ve always been using Mac for programming since my first job. I don’t actually own one myself because the company always give me one. But I just quit my last job and I’m broken af. So back to my old friend windows I guess! Anyway here are the steps:

WSL(Windows Subsystem for Linux) & Hyper (a fancy terminal)

  • Install WSL, You can watch this youtube video

  • Install choco

  • Install hyper

After this, you may want to use zsh and switch hyper’s default shell to zsh

Zsh

  • Install zsh & oh-my-zsh
  • Set zsh as default shell in WSL (link)

You may want to know where the hyper configuration file is:

Shadowsocks & Proxychains

For people like me in China, we all need this one more step…

  • Get yourself a shadowsocks server

  • Install shadowsocks client on ubuntu

  • Install and use proxychains (link)

Emacs25 & VcXsrv

  • Install emacs 25 on ubuntu (link)
  • Install spacemacs
  • If you use magit and on ubuntu 14.04, you need to upgrade git (link)
  • Install VcXsrv unless you just use emacs inside the terminal. As for me, I need GUI for Nyan cat
  • I like Firacode font, so I also installed it
  • Open XLauncher on windows with display number being 0
  • Set this inside WSL: export DISPLAY=localhost:0.0
  • And launch emacs

It’s not perfect. It’s slow af. The text is blurry and Chinese characters are a mess. But I think I can take it after all this trouble.

Update 1:

I can’t visit my local hexo server through 127.0.0.1:4000. After searching for a while, It turns out that it’s wegame’s fault (link). To solve this, you need to delete %systemroot%\system32\drivers\QMTgpNetflow764.sys and restart your computer. Or just uninstall wegame.
